# CLI 자바 게시판 서비스

---

## 🎯 프로젝트 소개

Java 콘솔 기반으로 동작하는 간단한 텍스트 게시판 프로그램입니다.

사용자의 명령어 입력을 통해 게시글을 생성, 조회, 수정, 삭제할 수 있으며
Java 기본 문법, 객체지향 설계, 리스트 자료구조 활용을 연습하는 것을 목표로 합니다.

---

## 🧩 주요 기능

| 기능        | 설명             |
| ----------- | ---------------- |
| write       | 게시글 작성      |
| list        | 게시글 목록 조회 |
| detail [id] | 게시글 상세 조회 |
| update [id] | 게시글 수정      |
| delete [id] | 게시글 삭제      |
| exit        | 프로그램 종료    |

---

## ⚙️ 실행 방법

프로그램 실행 후 콘솔에 명령어를 입력하여 기능을 사용할 수 있습니다.

```bash
명령어: write
명령어: list
명령어: detail 1
명령어: update 1
명령어: delete 1
명령어: exit
```

---

## 💬 실행 예시

```text
== 게시판 프로그램 ==

명령어: write
제목: 자바 공부
내용: CLI 게시판 만들기
=> 게시글이 등록되었습니다.

명령어: write
제목: 스프링 공부
내용: 기초부터 시작하기
=> 게시글이 등록되었습니다.

명령어: list
번호 | 제목 | 등록일
-------------------------
2 | 스프링 공부 | 2026-04-16
1 | 자바 공부 | 2026-04-16

명령어: detail 1
번호: 1
제목: 자바 공부
내용: CLI 게시판 만들기
등록일: 2026-04-16

명령어: update 1
새 제목: Java 공부
새 내용: 다시 정리중
=> 1번 게시글이 수정되었습니다.

명령어: delete 1
=> 1번 게시글이 삭제되었습니다.

명령어: exit
프로그램을 종료합니다.
```

## 🏗️ 프로젝트 구조

```
src/
└─ main/
   └─ java/
      └─ org/example/
         ├─ Main.java
         ├─ App.java
         ├─ Article.java
         └─ Rq.java
```

---

## 🧠 클래스 역할

### Main.java

- 프로그램 실행 진입점

### App.java

- 전체 프로그램 흐름 제어
- 명령어 처리 및 기능 실행

### Article.java

- 게시글 데이터 객체 (id, title, content, regDate)

### Rq.java

- 사용자 명령어 해석 (action + id 분리)

---

## 🎯 개발 포인트

- Scanner를 활용한 사용자 입력 처리
- ArrayList를 이용한 데이터 저장 및 관리
- 객체지향 설계를 통한 역할 분리
- 메서드 단위 기능 분리로 가독성 향상
- 명령어 파싱을 통한 CLI 구조 구현

---

## ⚠️ 예외 처리

- 존재하지 않는 게시글 번호 입력 시 안내 메시지 출력
- 잘못된 명령어 입력 시 안내 메시지 출력
- 숫자가 아닌 id 입력 시 예외 처리

---

## ✅ 느낀 점
