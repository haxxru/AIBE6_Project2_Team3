# CLI 자바 게시판 서비스

---

## 📌 프로젝트 소개

Java 콘솔 기반으로 구현한 간단한 게시판 프로그램입니다.

사용자는 명령어를 입력하여 게시글을 작성하고, 목록을 조회하고, 상세 내용을 확인하고, 수정 및 삭제할 수 있습니다.
또한 게시글 상세 조회 시 조회수가 증가하도록 구현하여 게시판의 기본 동작을 연습할 수 있도록 구성했습니다.

이 프로젝트를 통해 Java 기본 문법, 객체지향 설계, 계층 분리, 사용자 입력 처리, 리스트 기반 데이터 관리 방식을 연습할 수 있습니다.

---

## 🧩 주요 기능

| 기능          | 설명        |
| ----------- | --------- |
| write       | 게시글 작성    |
| list        | 게시글 목록 조회 |
| detail [id] | 게시글 상세 조회 |
| update [id] | 게시글 수정    |
| delete [id] | 게시글 삭제    |
| exit        | 프로그램 종료   |

## 🔗 클래스 역할

### Main.java

프로그램의 시작 지점입니다.

### App.java

프로그램 실행 루프를 담당하며, 사용자 명령어를 입력받아 각 컨트롤러로 전달합니다.

### Article.java

게시글 데이터 객체입니다.
게시글 번호, 제목, 내용, 작성일, 조회수를 관리합니다.

### ArticleRepository.java

게시글 데이터를 저장하고 조회하는 저장소 역할을 합니다.
`ArrayList<Article>`를 사용하여 CRUD를 처리합니다.

### ArticleService.java

게시글 작성, 수정, 삭제, 조회수 증가 등 비즈니스 로직을 담당합니다.

### ArticleController.java

게시글 관련 명령어를 처리하며, 사용자 입력과 출력 흐름을 담당합니다.

### SystemController.java

시스템 종료와 같은 공통 시스템 기능을 담당합니다.

### Rq.java

사용자가 입력한 명령어를 해석하여 action과 id를 분리하는 역할을 합니다.

## 💬 실행 예시

```text
== 게시판 프로그램 ==

명령어: write
제목: 자바 공부
내용: CLI 게시판 만들기
=> 1번 게시글이 등록되었습니다.

명령어: write
제목: 스프링 공부
내용: MVC 구조 이해하기
=> 2번 게시글이 등록되었습니다.

명령어: list
번호 | 제목 | 등록일 | 조회수
-----------------------------------
2 | 스프링 공부 | 2026-04-17 | 0
1 | 자바 공부 | 2026-04-17 | 0

명령어: detail 1
번호: 1
제목: 자바 공부
내용: CLI 게시판 만들기
등록일: 2026-04-17
조회수: 1

명령어: update 1
새 제목: Java 공부
새 내용: 콘솔 게시판 다시 정리중
=> 1번 게시글이 수정되었습니다.

명령어: delete 2
=> 2번 게시글이 삭제되었습니다.

명령어: exit
프로그램을 종료합니다.
```

## ✨ 개발 포인트

* Scanner를 활용하여 명령어와 사용자 입력을 처리했습니다.
* ArrayList를 활용하여 게시글 목록을 메모리에서 관리했습니다.
* Controller, Service, Repository 구조로 역할을 분리하여 가독성과 유지보수성을 높였습니다.
* 게시글 상세 조회 시 조회수가 1씩 증가하도록 구현했습니다.
* 목록 조회 시 최신 게시글이 먼저 보이도록 역순 출력했습니다.

---

## ⚠️ 예외 처리

* 존재하지 않는 게시글 번호를 입력하면 안내 메시지를 출력합니다.
* 숫자가 아닌 잘못된 게시글 번호 입력 시 예외를 방지하도록 처리했습니다.
* 존재하지 않는 명령어 입력 시 안내 메시지를 출력합니다.
* 빈 입력은 무시하고 다시 명령어를 입력받도록 처리했습니다.
